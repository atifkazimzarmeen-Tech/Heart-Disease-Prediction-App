# Heart Disease Prediction Web App

A **Flask-based web application** that predicts the risk of heart disease based on patient data using machine learning models including **KNN, Random Forest, Extra Trees**, and a **Voting Classifier** ensemble.

---

## Features

- User-friendly hospital-style web interface.
- Predict heart disease using patient details:
  - Age, Sex, Chest Pain Type, Resting BP, Cholesterol, Fasting Blood Sugar
  - Resting ECG, Max Heart Rate, Exercise Angina, Oldpeak, ST Slope
- Probability-based prediction with color-coded results.
- Machine learning models used:
  - K-Nearest Neighbors (KNN)
  - Random Forest
  - Extra Trees
  - Voting Classifier (ensemble)
- Responsive design suitable for desktop and mobile.
- Professional hospital dashboard look with sidebar navigation.

---

## Technologies Used

- **Frontend:** HTML, Bootstrap 5
- **Backend:** Python, Flask
- **Machine Learning:** scikit-learn (KNN, RandomForest, ExtraTrees, VotingClassifier)
- **Model Persistence:** Joblib
- **Data:** Heart Failure Dataset (CSV)
